=========== Nancy Tetris 说明文档 =================

1. src里源代码，运行需要安装pygame库
2. NancyTeris目录中为编译的exe版本，可直接运行tetris.exe


Nancy Tetris 说明
---------------------------------------------------
1. 单击游戏  -- 直接运行 tetris.exe
2. 联机对战（只支持两人对战）
   2.1 主机加命令行参数运行：
       tetris.exe server
   2.2 另一台机器加主机IP命令行参数运行：
       tetris.exe 192.168.58.XX


操作说明
--------------------------------------
-----       ↑    变形          ------
-----       ←    左移          ------
-----       →    右移          ------
-----       ↓    加速          ------
-----      Space  直接掉落      ------
-----      Enter  重新开始      ------
